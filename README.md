# 🧮 Reverse Polish Notation (RPN) Calculator

## Deskripsi
Aplikasi kalkulator Reverse Polish Notation (RPN) berbasis web yang dibuat menggunakan Python dan Streamlit.

Program ini menerapkan konsep Struktur Data Stack untuk melakukan perhitungan matematika menggunakan notasi postfix (RPN).

## Fitur

- Penjumlahan (+)
- Pengurangan (-)
- Perkalian (*)
- Pembagian (/)
- Pangkat (^)
- Akar Kuadrat (sqrt)
- Sinus (sin)
- Cosinus (cos)
- Tangen (tan)
- Logaritma (log)
- Riwayat Perhitungan
- Tampilan Modern Berbasis Streamlit

## Cara Menjalankan

### 1. Install Dependency

```bash
pip install -r requirements.txt
```

### 2. Jalankan Program

```bash
streamlit run app.py
```

## Contoh Penggunaan

### Penjumlahan

Input:

```text
3 4 +
```

Output:

```text
7
```

### Perkalian

Input:

```text
5 6 *
```

Output:

```text
30
```

### Pangkat

Input:

```text
2 3 ^
```

Output:

```text
8
```

### Akar Kuadrat

Input:

```text
9 sqrt
```

Output:

```text
3
```
